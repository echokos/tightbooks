## About

Tight Books is a private fork of [BigCapital](https://github.com/bigcapitalhq/bigcapital), an open-source accounting and inventory platform. It provides double-entry bookkeeping, automated accounting processes, and financial statements and reports.

> Tight Books is a fork of [BigCapital](https://github.com/bigcapitalhq/bigcapital), licensed under AGPL-3.0.

## Getting Started

### Self-hosted (Docker)

Tight Books runs via Docker Compose. See the [upstream Docker guide](https://docs.bigcapital.app/deployment/docker) for reference configuration.

### Local Development

See the [upstream contributing guide](https://github.com/bigcapitalhq/bigcapital/blob/develop/CONTRIBUTING.md) for local setup instructions.
